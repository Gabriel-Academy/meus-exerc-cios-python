### Exercício 033
* **O que foi feito?** Um identificador de números maiores e menores.

**2°** "if" SE "c" for menor "<" que "a" e "and" "c" for menor "<" que "b" logo "c" é o menor.
**3°** Definindo "a" como MAIOR "if" SE "b" for maior ">" que "a" e "b" for maior ">" que "c" logo "b" é o maior.
**4°** "if" SE "c" for maior ">" que "b" e "c" maior ">" que "a" logo o maior é "c".
